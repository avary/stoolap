Why Stoolap?

Modern database design for today's application demands

High Performance

Columnar storage and vectorized execution deliver analytics at incredible speed.

Memory-First

Optimized for in-memory operation with optional persistence for durability.

Columnar Storage

Data organized by column for efficient compression and blazing-fast analytics.

MVCC Transactions

SQL Standard

Get Started in Minutes

Simple to install, easy to use

terminal
$ go get github.com/stoolap/stoolap
$ stoolap
Stoolap Database Server v1.0.0
Connected to memory database
stoolap> CREATE TABLE users (id INT PRIMARY KEY, name VARCHAR(255));
Table created successfully.
stoolap> INSERT INTO users VALUES (1, 'John Doe');
1 row affected.
stoolap> SELECT * FROM users;
+----+----------+
| id | name |
+----+----------+
| 1 | John Doe |
+----+----------+
1 row in set.
